# 致命的交易习惯

1. 舍不得止损
1.1 基本功 任何好的进攻都是建立在防守之上 遇到单边行情做反方向必爆仓
1.2 严格执行交易计划 盈利和亏损才能游刃有余

2. 频繁交易
2.1 手续费
2.2 行情把握不住 影响心态 

3. 过度关注盈亏本身
3.1 真正在看待自己仓位的时候 反而不会过多关注行情变化 关注点更多在盈亏本身
3.2 会打乱交易计划

4. 什么行情都想做
4.1 看准趋势做单 而不是波段
4.2 等行情走出来 有些行情就是吃人

5. 过度自信 不冷静
5.1 做好仓位管理
5.2 挣钱需要保持冷静 不要上头
5.3 用闲钱做单

6. 无用消息过多
6.1 消息太杂 影响判断

7. 多交易所 多分析师
7.1 交易所跑路
